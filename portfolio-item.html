Animatedcoin flip app!

Using a romanian coin

This project was so much fun!

The idea of this app came one day when I had to make a decision and wanted to flip a coin to make up my mind (very mature, right?). The problem was that at that moment, I had none left in my pocket. So, I thought for a second, why not have an app for this kind of "problem"?

This project provided an opportunity to showcase my CSS and JavaScript skills. To develop this app, I used two pictures(of a Romanian coin), rotated them with CSS, and achieved the final result by using JavaScript. Also, took the chance provided by the opportunity and designed the buttons as well!

At the moment, the app is only pulling data from an API and displays a random advice, but for the future, I am working on implementing some more features, so make sure you stay tuned!
